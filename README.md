# yup-ls

```
NAME:
   ls - list directory contents

USAGE:
   ls [OPTIONS] [FILE...]

      List information about the FILEs (the current directory by default).

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --long, -l        use a long listing format (default: false)
   --all, -a         do not ignore entries starting with . (default: false)
   --human-readable  with -l, print sizes in human readable format (default: false)
   --recursive, -R   list subdirectories recursively (default: false)
   --reverse, -r     reverse order while sorting (default: false)
   --sort value      sort by WORD (name, time, size) (default: "name")
   --help, -h        show help
```
